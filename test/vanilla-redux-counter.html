A counter using a vanilla Redux store

The original code for this page can be found in the article How Redux works ? (Only HTML & Pure JS).
I have modified it as follows:

  1. Split the HTML and the javascript code into separate files.
  2. Reformat the page by adding a heading, this list of changes and a few layout tweaks.
  3. Move the script tag for the redux code from the end of the body into the head, which necessitates the following change...
  4. Wrap all of the initialization code into a single function and defer its execution until the window load event.
  5. Add a new button and corresponding action to square the number in the store.
  6. Log unexpected actions to the console to help with debugging the addition of future actions.
  7. Enable the Redux Devtools plugin if it's installed in the browser.

The stored value is 0.
